# QR Code Generator 📱

A simple and responsive **QR Code Generator** built using **HTML, CSS, and JavaScript**.  
This project allows users to generate QR codes from text or URLs and automatically download them as a **PDF file**.

---

## 🚀 Features

- Generate QR codes instantly
- Convert QR code into downloadable PDF
- Clean and responsive UI
- Input validation
- Supports text, links, and URLs
- Built using pure HTML, CSS, and JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- QRCode.js Library
- jsPDF Library

## 📜 Required CDN Links

Add these CDN links before closing the `</body>` tag in `index.html`.

### QRCode.js

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js"></script>
```

### jsPDF

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js"></script>
```

---

## 💻 How It Works

1. User enters text or URL
2. JavaScript generates a QR code
3. QR code is displayed on screen
4. The QR code is converted into an image
5. jsPDF creates a PDF containing the QR code
6. PDF automatically downloads
